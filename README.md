# posture-alignment-cv

Real-time posture monitoring via webcam and MediaPipe pose estimation. Detects forward head posture, slouching, and lateral lean, then alerts you with visual and audio notifications before discomfort sets in. All processing happens locally — no video data leaves your machine.
